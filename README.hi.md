## हिन्दी विवरण (Hindi Specification)

AnonVault एक अत्यधिक सुरक्षित, गोपनीयता-प्रथम व्यक्तिगत कार्यक्षेत्र है जिसे हैकाथॉन इवेंट्स को ट्रैक करने, दैनिक कार्यों को प्रबंधित करने, नवीन प्रोजेक्ट विचारों का मंथन करने और रचनात्मक विचारों को सुरक्षित रखने के लिए डिज़ाइन किया गया है।

### डैशबोर्ड UI की खास बातें

- **हेडर**: बाईं ओर ग्रेडिएंट टाइटल। दाईं ओर तारीख, दिन और लाइव `HH:MM · SS` घड़ी एक ही पिल में।
- **कोट कार्ड**: सेरिफ़ फ़ॉन्ट (Georgia) में इटैलिक कोट, बड़े डेकोरेटिव `"` वॉटरमार्क के साथ।
- **साइडबार**: डैशबोर्ड नेविगेशन आइकन पर नीला **पिन बैज** — यह दर्शाता है कि यह मुख्य व्यू है।
- **सोशल लिंक्स**: साइडबार फुटर में X, GitHub और LinkedIn लिंक्स हूवर टूलटिप्स के साथ।

### कैसे शुरू करें

#### 1. डेटाबेस और स्टोरेज सेटअप
Supabase पर एक निःशुल्क प्रोजेक्ट बनाएं:
1. **SQL Editor** में [supabase/schema.sql](supabase/schema.sql) की स्क्रिप्ट चलाएं।
2. **Storage** पैनल में `idea-images` नाम का एक पब्लिक बकेट बनाएं।

#### 2. एनवायरनमेंट कॉन्फ़िगरेशन
प्रोजेक्ट रूट में `.env` फाइल बनाएं:
```env
VITE_SUPABASE_URL=आपका_supabase_प्रोजेक्ट_url
VITE_SUPABASE_ANON_KEY=आपका_supabase_anon_api_key
VITE_APP_PIN=आपका_4_अंकों_का_सिक्योर_पिन
```

#### 3. लोकल सर्वर चलाएं
```bash
npm install
npm run dev
```
ब्राउज़र में [http://localhost:5173](http://localhost:5173) खोलें।
